# **App Name**: MediSummarize

## Core Features:

- Audio Capture: Audio Recording: Capture spoken conversations during medical consultations.
- AI Transcription: Transcription: Transcribe recorded audio into text using AI.
- AI Summarization: Summarization: Summarize transcribed text using GPT to extract key information.
- Summary Display: Summary Display: Present summarized medical information in a clear and concise format.
- User Authentication: User Authentication: Secure user accounts to protect sensitive medical information.

## Style Guidelines:

- Primary color: Calming blue (#E3F2FD) for a professional and trustworthy feel.
- Secondary color: Light gray (#FAFAFA) for backgrounds and subtle elements.
- Accent: Teal (#009688) for interactive elements and highlights.
- Clean and readable fonts for medical terms and summaries.
- Use clear and universally understood medical icons.
- Clean and well-organized layout to prioritize information clarity.
